# Fonts for Pomodoro Timer

This directory contains font files used by the Pomodoro Timer application, particularly for the Three.js 3D timer visualization.

## Required Fonts

- `Inter-Bold.woff`: Used for the timer display
- `Inter-Regular.woff`: Used for status text

## Adding Custom Fonts

To add custom fonts, place the font files in this directory and update the references in the `ThreeJsTimer.tsx` component.
