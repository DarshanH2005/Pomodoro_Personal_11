# Sound Effects for Pomodoro Timer

This directory contains sound effects used by the Pomodoro Timer application.

## Sound Files

- `work-complete.mp3`: Played when a work session is completed
- `break-complete.mp3`: Played when a break session is completed
- `button-click.mp3`: Played when buttons are clicked
- `timer-tick.mp3`: Played for timer tick sounds

## Adding Custom Sounds

To add custom sounds, place MP3 files in this directory and update the `soundEffects.ts` file in the `lib` directory.
